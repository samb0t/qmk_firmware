This layout adds a new function layer similar to the default one from the pok3r:

    /* Function
     * ,-----------------------------------------------------------------------------------.
     * | Esc  |      | Prev | Play | Next |      |      |Pg Up | Home |Pg Dn |Prt Sc| Del  |
     * |------+------+------+------+------+-------------+------+------+------+------+------|
     * |      |      | Vol- | Mute | Vol+ |      | Left | Down |  Up  |Right | End  |      |
     * |------+------+------+------+------+------|------+------+------+------+------+------|
     * |      |      |Bcklit|      |      |      |      |      | End  |      |      |      |
     * |------+------+------+------+------+------+------+------+------+------+------+------|
     * |      |      |      |      |      |             |      | Home |Pg Up |Pg Dn | End  |
     * `-----------------------------------------------------------------------------------'
     */

You can acces this layer by holding the bottom-left key.
